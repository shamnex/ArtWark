Quick Setup
### 1. Install required dependency with composer
       composer install
### 2. configure your .env file with database config.
### 3. migrate database schema
       php artisan migrate
### 4. run database seeder to populate the database
       php artisan db:seed
### 5. run development server
       php artisan serve
